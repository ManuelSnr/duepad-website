# Duepad — Shared subscriptions, sorted before they're due

A landing page for **Duepad**, an app that takes the chaos out of managing shared subscriptions with a group.

## 🌐 Live Preview

> Coming soon — deploy via GitHub Pages or Netlify.

## 📋 About

Duepad solves a common pain: when you share a subscription (Netflix, Spotify, ChatGPT, etc.) with friends or family, one person always ends up doing all the admin work — reminding everyone, chasing confirmations, recalculating splits, and sharing login details manually.

Duepad automates all of that.

## ✨ Features

- **Automatic reminders** — Everyone in the group gets notified 7 and 2 days before renewal
- **"Still in?" poll** — One-tap opt-in so everyone declares their intent before renewal
- **Auto split calculator** — Cost per person updates instantly when someone opts out
- **Secure access details** — Credentials stored once, visible only to confirmed members
- **Shared dashboard** — Everyone sees upcoming renewals, confirmations, and what they owe

```bash
open index.html
```
